# Nelson Logic

Portfolio and project intake for Nelson Logic — *Fluid Intelligence*. Built with Next.js, Tailwind CSS, and Framer Motion.

## Routes

- **/** — Home: hero (aurora gradient), selected deployments, services, intake form
- **/crave** — Crave dashboard (Dish.io Intelligence)

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech

- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- Formspree (intake form)
- Lucide React
